# 🍧 GL Smart Search SDK Cookbook

Welcome to the **GL Smart Search SDK Cookbook** - a comprehensive collection of minimal, standalone cookbook examples for working with the GL Smart Search SDK.

## 🚀 Quick Start

1. **Install Python 3.11+** and **[uv](https://docs.astral.sh/uv/)**

2. **Navigate to an example directory:**
   ```bash
   cd examples/web          # For web search examples
   # or
   cd examples/connector    # For connector examples
   ```

3. **Follow the README in that directory** for setup and usage instructions.

## 📁 Examples

- **[Web Search Examples](examples/web/)** - Web search functionality cookbook
- **[Connector Examples](examples/connector/)** - Connector integration cookbook

Each example directory contains its own README with detailed documentation, setup instructions, and usage examples.
